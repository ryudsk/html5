html5 시멘틱 구조태그

1. header

website의 상단영역으로 Logo, 검색폼, 메인메뉴(navigation) 등의 요소를 포함

2. nav

navigation의 약자로 메뉴목록 영역
보통 메인메뉴(gnb-Global Navi Bar), 서브메뉴(lnb-Local Navi Bar) 등을 표현할 때 사용

3. section

의미가 같은 내용 끼리의 묶는 contents 그룹요소(예를 들면 신문의 정치, 경제, 사회, 문화 등)
section 요소 사용 시 반드시 바로 아래 제목요소(h2~h6) 중 하나가 나와야 함 - 웹표준 마크업 검사 시 체크항목

4. article

독립적인(자립적인) contents - h2~h6

5. aside

웹사이트 측면(left, right)에 들어가는 사이드메뉴, 배너 등

6. footer

웹사이트 하단영역, 로고, 회사명, 주소, 메뉴, 카피라이트 등

새로 추가된 태그요소

figure, figcaption

mark

웹접근성이란? 누구나 신체적 환경적 조건에 상관없이 웹에 접근하여 사용할 수 있도록 보장하는 것을 말한다. 검색 키워드 등을 강조 처리

time

사용자가 작성한 날짜나 시간을 브라우저가 이해하도록 표시

오전 10시 52분 2017년 12월 24일
